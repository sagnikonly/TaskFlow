## 🎨 Color Legend

### Status Colors
- 🟢 **Green**: Success, on track, positive
- 🟡 **Yellow/Amber**: Warning, needs attention
- 🔴 **Red**: Alert, action required
- 🔵 **Blue**: Information, neutral
- 🟣 **Purple**: Primary actions, features

### Metric Colors
- **Violet**: Completed tasks
- **Pink**: Weekly stats
- **Cyan**: Step-ups
- **Green**: Success rates
- **Orange**: Active items
- **Amber**: AI features

## 🎯 Quick Tips

### Navigation
- **Tap cards** to open detailed views
- **Back button** (←) to return
- **Smooth animations** guide your flow

### Understanding Metrics
- **Percentages**: Completion rates
- **Streaks**: Consecutive days
- **Trends**: ↑ up, ↓ down, → stable
- **Bars**: Visual comparisons

### Best Practices
1. Check analytics **weekly**
2. Act on **AI recommendations**
3. Celebrate **milestones**
4. Adjust **goals** based on insights
5. Use **heatmap** for planning

---

## 🚀 Getting Started

1. **Complete some tasks** to generate data
2. **Open Analysis** from bottom nav
3. **Explore each card** to see insights
4. **Set your goal** in Settings
5. **Track progress** regularly

## 💡 Pro Tips

### Maximize Insights
- Complete tasks consistently for better trends
- Use different priorities to see patterns
- Set realistic goals for accurate projections
- Review AI suggestions weekly
- Filter heatmap by subject for focus

### Optimize Productivity
- Schedule tasks on your best days
- Follow optimal task load suggestions
- Watch for burnout warnings
- Celebrate milestone achievements
- Use step-ups to grow gradually
